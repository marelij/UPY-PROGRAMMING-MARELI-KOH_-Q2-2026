ALGORITHM Selection_Sort_Animated

## Generar una lista de números aleatorios
numbers ← lista de N números aleatorios

## Configurar la ventana de dibujo
Configurar escala X
Configurar escala Y

## Obtener el tamaño de la lista
n ← longitud(numbers)

## Recorrer toda la lista
FOR i ← 0 TO n - 1 DO

    ## Suponer que el primer elemento es el mínimo
    min ← i

    ## Buscar el elemento más pequeño en el resto de la lista
    FOR j ← i + 1 TO n - 1 DO

        ## Dibujar las barras resaltando el mínimo y el elemento actual
        Draw_Bars(numbers, min, j)

        ## Comparar los valores
        IF numbers[j] < numbers[min] THEN
            min ← j
        END IF

    END FOR

    ## Si se encontró un nuevo mínimo, intercambiar posiciones
    IF min ≠ i THEN

        temp ← numbers[i]
        numbers[i] ← numbers[min]
        numbers[min] ← temp

        ## Mostrar el intercambio en la animación
        Draw_Bars(numbers, i, min)

    END IF

END FOR

## Dibujar la lista completamente ordenada
Draw_Bars(numbers)

## Mostrar el resultado final
Display_Window()

END ALGORITHM
