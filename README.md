## CW07 - Verification Digit Calculator

The included Python script calculates the UTFSM verification digit of a given rol number following these steps:

- Reverses the rol number
- Multiplies each digit by the sequence 2,3,4,5,6,7
- Applies modulo 11
- Subtracts the result from 11
