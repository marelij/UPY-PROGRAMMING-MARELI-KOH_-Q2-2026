## Classwork #10 - School Management System
Description
Python program that simulates a school management system with three user roles: Student, Professor, and Coordinator. Each role has different permissions and access to academic information.

# Student Role
Views academic report

Sees all subject grades

Calculates approved and pending subjects

# Professor Role
Views list of students

Selects a student and subject

Updates grades with validation and confirmation

# Coordinator Role
Views list of professors

Views all subjects

Views full grade report of all students

## How to Run
Ensure you have Python 3 installed on your system. Clone this repository or download the source file:
